## 🕹 사용방법

```
  1. git clone https://github.com/rnfltpgus/nitend.git
  2. cd nitend
  3. npm install or npm i
  4. cd ios
  5. pod install
  6. cd ..
  7. npm start
  8. npm run ios
```
